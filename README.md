# 🛒 Ecommerce Sales Dashboard – Data Analytics Project
An interactive Excel-based Ecommerce Sales Dashboard designed to analyze sales performance, customer behavior, product trends, and revenue insights. This project demonstrates strong skills in data cleaning, transformation, visualization, and KPI reporting using Excel.
# 🚀 Project Overview
This dashboard provides a comprehensive view of ecommerce performance by tracking:

Total Sales & Revenue Trends

Top-Selling Products

Category-wise Performance

Region-wise Sales Distribution

Customer Purchase Patterns

Monthly & Quarterly Sales Trends

Profitability Insights

# 📊 Key Features
Dynamic Pivot Tables and Pivot Charts
Interactive Slicers for filtering by:

Region

Category

Month

Customer Segment

Cleaned and transformed raw data using:

Power Query

Excel formulas (VLOOKUP, IF, TEXT, DATE functions)

KPI Cards for:

Total Revenue


Total Orders

Average Order Value

Profit Margin

Trendline charts for monthly performance

Heatmaps for category-wise sales comparison

# 🧹 Data Cleaning & Preparation
Performed using Excel & Power Query:

Removed duplicates and null values

Standardized date formats

Split and merged columns

Created calculated fields (Profit, AOV, Discount %)

Normalized product categories

Ensured consistent naming conventions

Built a structured data model for dashboard use

# 📈 Dashboard Insights
Some of the insights the dashboard reveals:

Top-performing product categories and their contribution to total revenue

Seasonal trends showing peak sales months

Regions with highest customer demand

High-value customers and repeat purchase patterns

Products with low performance that may require marketing or discounting
